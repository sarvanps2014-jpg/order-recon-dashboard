This project uses a dual-repository approach:

1. **IBM GitHub (Primary)** - For code backup and version control

   - All commits use IBM email: sarvan_ps@in.ibm.com
   
2. **Public GitHub (Live)** - For dashboard hosting
   - Repository: https://github.com/sarvanps2014-jpg/order-recon-dashboard

Use `UPDATE_DASHBOARD_DUAL.bat` to push to both repositories simultaneously.
